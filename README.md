# Springboot - Vue projects
This is a practice project based on Java, Vue, Springboot, MyBatis as the tech stack. For a front-end and back-end isolated project as the practice project after finishing the learning of the springboot framework. 

# Start the project

## Front-end Vue project

Use command 

```
npm run dev
```

## Back-end Springboot project

Go to IDEA and click run at the configuration main class
